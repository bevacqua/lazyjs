Production - Build Process

  1. A CLI tool minifies and bundles script files
  2. Individual modules are identified with lazy define comment directives

Page Load Flow

  1. You reference LazyJS
  2. You reference the bundle created by the CLI
  3. LazyJS fetches the bundle as soon as it executes
  4. LazyJS doesn't parse individual modules until they're needed
  5. LazyJS evaluates the jumpstart script
  6. No further AJAX is necessary
  7. Scripts are evaluated on demand
  8. You sit back and enjoy
